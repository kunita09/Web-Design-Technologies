เทคนิคการออกแบบเว็บเพจให้แสดงผลได้เร็ว

ดวงพร เกี๋ยงคำ และวงศ์ประชา จันทร์สมวงศ์ (2546) กล่าวไว้ว่า เทคนิคการออกแบบเว็บเพจให้แสดงผลได้เร็ว มีหลายวิธี เช่น

1. ลดขนาดของไฟล์ HTML โดยกำจัดคำสั่ง หรือแท็ก tag ที่ไม่จำเป็นออกไป
2. ใช้ตัวอักษร และลิงค์แบบ html
3. ใช้ graphic ให้ถูกประเภท
4. ตัดแบ่งภาพที่มีขนาดใหญ่ให้เป็นภาพชิ้นเล็กๆ
5. นำกราฟิกกลับมาใช้ใหม่ในหน้าอื่นๆ
6. กำหนดค่า Width และ Height ของกราฟิก
7. กำหนดค่า Alt ของกราฟิก
8. ใช้ style sheet ในการจัดรูปแบบ
